To run portscan:

nmap -sT -sU -p- 192.168.1.4

UDP:
sudo nping --udp -p 53 --rate 10000 --count 0 192.168.1.4

SYN:
sudo nping --tcp -p 80 --flags SYN --rate 10000 --count 0 192.168.1.4

ICMP:
ping -f 192.168.1.4

DNS AMPLIFICATION:
sudo nping --udp -p 53 --data-string "custom-DNS-query-payload" --source-ip 192.168.1.2 --count 500 --rate 1000 192.168.1.4
